◆ printTable()

void printTable ( const vector< worker > & data)

Выводит таблицу с данными о сотрудниках в консоль.

Данная функция принимает вектор структур worker, который содержит информацию о сотрудниках, их отработанном времени по месяцам, и выводит эти данные в консоль в виде таблицы. Таблица состоит из трех столбцов: "Имя сотрудника", "Месяц" и "Отработанное время". Время работы сотрудников выводится в формате "дни:часы:минуты". Месяцы отображаются на русском языке с использованием заранее подготовленного словаря.

Аргументы
dataВектор структур worker, содержащий:
  • имя сотрудника (name);
  • список месяцев с отработанным временем (work_time), где каждый месяц представлен числом, а время работы — количеством минут.

◆ readData()

vector< worker > readData ( const string & file_name)

Читает данные из файла и преобразует их в вектор структур worker.

Данная функция обрабатывает текстовый файл, содержащий записи о сотрудниках, их рабочем времени и дате. Каждая строка файла содержит дату, имя сотрудника, время начала и окончания работы. Данные преобразуются в вектор структур worker, где для каждого сотрудника сохраняется общее рабочее время по месяцам.

Аргументы
file_nameСсылка на строку с именем файла для чтения.
Возвращает
Вектор структур worker, содержащий:
  • имя сотрудника (name);
  • список месяцев с общим количеством отработанных минут (work_time).
Исключения
runtime_errorЕсли файл не удается открыть.

const vector< worker > & data )

Записывает данные о сотрудниках в CSV файл.

Данная функция принимает вектор структур worker, который содержит информацию о сотрудниках, их отработанном времени по месяцам, и записывает эти данные в CSV файл. Каждый сотрудник записывается в отдельной строке, в которой указывается имя сотрудника, месяц и отработанное время. Время работы преобразуется в формат "дни:часы:минуты" с помощью вспомогательной функции. Месяцы отображаются на русском языке с использованием заранее подготовленного словаря.

Аргументы
file_nameСсылка на строку с именем файла для записи данных.
dataВектор структур worker, содержащий:
  • имя сотрудника (name);
  • список месяцев с отработанным временем (work_time).
Исключения
runtime_errorЕсли файл не удается открыть для записи.
