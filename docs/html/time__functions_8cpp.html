◆ dateToMonth()

short dateToMonth ( string date)

Извлекает номер месяца из строки с датой в формате DD-MM-YYYY.

Данная функция разбивает строку с датой по разделителю - и извлекает второе поле, соответствующее номеру месяца (MM). Предполагается, что входная строка имеет корректный формат (например, "22-01-2453").

Аргументы
dateСтрока с датой в формате DD-MM-YYYY.
Возвращает
Номер месяца (целое число от 1 до 12).
